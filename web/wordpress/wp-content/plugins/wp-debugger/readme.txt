A plugin that helps you locate and fix issuess with your Wordpress installation.

== Installation ==

Simply extract the contents of the plugin zip file to a folder named wordpress-debugger and place it in your ./wp-content/plugins folder. Then log in to your Wordpress admin panel and activate the plugin. You will then get a new menu element in the Tools section of the admin panel.
